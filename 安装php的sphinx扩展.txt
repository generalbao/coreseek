可以在这里（http://pecl.php.net/package/sphinx）找到sphinx的php扩展源码 下载

注意，使用phpize，configure的时候可能会要求要安装libsphinxclient，它在coreseek-xxx
/csft-xx/api/libsphinxclient/里面能找到，编译安装它以后就可以configure，make，生成动态so文件了。


11111
cd libsphinxclient/
./configure
make && make install


22222

cd sphinx
/usr/bin/phpize

./configure --with-php-config=/usr/bin/php-config --with-sphinx(如果没安装libshinxclient就会报错)
